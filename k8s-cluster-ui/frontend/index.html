Getting Started

To get started with K8s Clusters Dashboard:

  1. Choose the cloud provider dashboard you want to use (GKE or EKS)
  2. Connect to your existing clusters or discover available clusters
  3. View resources and monitor your Kubernetes environment

For more information on how to use this dashboard, please refer to the User Guide.
