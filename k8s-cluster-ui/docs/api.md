# K8s Cluster UI - API Documentation

This document provides details about the API endpoints available in the K8s Cluster UI application.

## Base URL

```
http://localhost:5000/api
```

## Endpoints

### Health Check
- **URL**: `/health`
- **Method**: `GET`
- **Response**:
  ```json
  {
    "status": "ok",
    "message": "API is running"
  }
  ```
